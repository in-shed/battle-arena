# Top-Down Game (Godot)

A simple top-down game built with godot. 

---

## Features

- Top-down movement
- Tilemap-based levels
- Inventory
- Equipment and consumable items
- Enemy AI
- Combat system (melee / ranged)

---
## Controls
| Action         | Key                    |
|----------------|------------------------|
| Move           | "W A S D"              |
| Melee attack   | "LMB"                  |
| Ranged attack  | "RMB"                  |
| Inventory      | "E"                    |
| Pause          | "Esc"                  |
